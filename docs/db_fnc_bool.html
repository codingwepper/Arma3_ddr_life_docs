db_fnc_bool

Handles bool conversion for MySQL since MySQL doesn't support 'true' or 'false' instead MySQL uses Tinyint for BOOLEAN (0 = false, 1 = true)

Parameters
INT/BOOLBOOL - Bool to convert
INTMode - [0: Convert BOOL to INT, 1: Convert INT to BOOL]
Returns
INT - Mode 0
BOOL - Mode 1
Example
[STRING,MODE,MULTIARRAY] call db_fnc_bool;
